# 🎓 Université Horizon - Site Web Institutionnel

## 📌 Description du projet
Ce projet est un site web institutionnel complet pour une université fictive appelée **Université Horizon**.  
Il a été développé dans le cadre d’un projet académique en utilisant HTML5, CSS3 et JavaScript.

Le site présente les informations essentielles d’une université moderne :
- Facultés et départements
- Processus d'admission
- Vie étudiante
- Recherche
- Formulaire de contact

---

## 🚀 Fonctionnalités

- ✔ Site multi-pages (7 pages)
- ✔ Design moderne et professionnel
- ✔ Navigation entre les pages
- ✔ Formulaires avec validation HTML5
- ✔ Tableaux HTML pour les frais académiques
- ✔ Section vie étudiante (clubs, sports, événements)
- ✔ Responsive design (mobile & tablette)
- ✔ Interface simple et intuitive
